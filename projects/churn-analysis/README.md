# User Engagement & Churn Analysis

## Problem Statement
User retention is critical for digital products. This project focuses on identifying
early indicators of user disengagement and predicting churn using data-driven methods.

## Objective
- Predict the likelihood of user churn
- Identify behavioral patterns contributing to disengagement
- Provide actionable insights for product improvement

## Approach
- Exploratory Data Analysis (EDA)
- Feature engineering on user activity metrics
- Machine learning model development
- Model evaluation using standard performance metrics

## Tools & Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## Outcome
The analysis highlights key factors influencing churn and demonstrates how data science
can guide product-level decisions.
